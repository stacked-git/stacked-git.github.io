Patch Inspection
~~~~~~~~~~~~~~~~

linkstg:diff[]::
    Show a diff
linkstg:files[]::
    Show files modified by a patch
linkstg:id[]::
    Print git hash of a StGit revision
linkstg:log[]::
    Display or optionally clear the stack changelog

    Print patch name of a StGit revision

    Show patch commits
